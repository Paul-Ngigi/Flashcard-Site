Javascript(JS)

Javascript is a programming language
often used for web interactivity.It is commonly referred to as a scripting language and it has its frameworks e.g jQuery and Angular

Operators

Operators are are constructs defined within programming languages which behave like functions. Common examples include arithmetic,comparison and logical Operators.

Variables

A variable is a symbol which functions as a placeholder for varying expression or quantities, and is often used to represent an arbitrary element of a set. In addition to numbers, variables are commonly used to represent vectors, matrices and functions.

Variable naming conventions

A variable naming convection is a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

Function

Functions are "self contained" modules of code that accomplish a specific task. Functions usually "take in" data, process it, and "return" a result. Once a function is written, it can be used over and over and over again.
